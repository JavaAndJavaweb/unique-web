Unique-Web是一个轻量级高性能web开发框架，暂时没有加入开源行列，部分功能未实现。
	包含的服务
		webmvc				|	基础的web控制器
		
		restapi				|	基于rest风格的url访问
		
		ioc					|	一个基础的ioc容器
		
		dbpool				|	数据库连接池插件
		
		basedao				|	基础的数据库访问插件
		
		cache-plugin		|	redis缓存插件
		
		image-plugin		|	图片处理插件（https://git.oschina.net/biezhi/image-plugin.git）
		
		patchca				|	多彩验证码插件	
			
		httl				|	httl模板引擎
